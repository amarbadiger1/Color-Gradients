# 🎨 Gradient Generator Dev Tool

A simple and powerful **gradient color generator** built for developers and designers to quickly create, preview, and use beautiful gradients in their projects.

> ⚡ Built for practice, learning, and improving frontend development skills.

---

## 🚀 Features

- 🎨 Generate linear & radial gradients  
- 📋 Copy CSS and tailwind Css gradient code instantly    

---

## 🛠️ Tech Stack

- React.js  
- Tailwind CSS  
- JavaScript (ES6+)  

---

## 💡 Use Cases

This tool can be used as a **developer utility** in many scenarios:

- 🎯 Quickly generate background gradients for websites  
- 🎯 Use gradients in UI components (buttons, cards, sections)  
- 🎯 Create CSS backgrounds without manually writing code  
- 🎯 Experiment with color combinations  
- 🎯 Speed up frontend development workflow
